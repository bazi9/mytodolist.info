# 📝 Online ToDo Manager

> A lightweight, secure, and fully responsive personal task management application built with Python. 

### 🚀 Live Preview

**Experience the live app here:** [https://www.mytodolist.info](https://mytodolist.info)

\---

## 📖 About The Project

This project is a comprehensive ToDo list application designed to help users organize their daily workflow, tackle large projects by breaking them down into subcategories, and visually track their progress. It features a secure user authentication system, ensuring that everyone's data remains private and personalized.

## ✨ Key Features

* **🔒 Secure User Accounts:** Personalized dashboards with registration and login functionality.
* **🎯 Subtask Management:** Break down large tasks into smaller, manageable subcategories.
* **📊 Visual Progress Tracking:** Dynamic completion meters and a dashboard chart (powered by Chart.js) that updates as you finish tasks.
* **🏷️ Custom Keywords/Tags:** Add specific keywords to subtasks to keep detailed notes organized.
* **🌓 Dark/Light Mode:** A built-in theme toggle for comfortable viewing at any time of day.
* **📱 Fully Responsive:** Built with Bootstrap 4 to work flawlessly on desktop, tablet, and mobile browsers.
* **🖱️ Drag \& Drop:** Easily reorder tasks and subtasks (powered by Sortable.js).

\---

## 🛠️ Built With

**Frontend:**

* HTML5 / CSS3
* JavaScript (Vanilla \& jQuery)
* [Bootstrap 4](https://getbootstrap.com/docs/4.0/) (UI Framework)
* [Chart.js](https://www.chartjs.org/) (Data Visualization)
* [Sortable.js](https://sortablejs.github.io/Sortable/) (Drag and drop interactions)
* FontAwesome (Icons)

**Backend:**

* Python
* SQLite3 (Lightweight database `todo.db`)
* Jinja2 (HTML Templating)
